Quest1. HTML 4.x 이후의 HTML 표준의 변천사는 어떻게 되나요?
- 인터넷의 빠른 발전으로 W3C에서는 HTML4 의 제한된 태그로는 웹의 흐름을 따라가기 힘들다고 여김
  공식적으로 HTML 표준 업데이트 하지 않겠다고 발표
- XHTML 이라는 새로운 웹 표준 개발 프로젝트를 시작하여,
2000년 XHTML1 을 표준안으로 발표하지만 사용이 어려워 환영받지 못함.
- 마이크로소프트를 제외한 애플, 모질라, 오페라, 구글 등
당시 주요 브라우저 업체들은 페이지의 기본 기능인 HTML 을 확장하여 동적인 웹 서비스를 구현하자는 의견과 함께
WHATWG(Web Hypertext Application Technologies Working Group)를 구성
- W3C 는 2007년 WHATWG 를 받아들여 공식적으로 HTML 워킹그룹을 만들어서,
'웹 애플리케이션1.0' 을 HTML5 라 부르게 되고,XHTML2 개발을 폐기
- 2014년 W3C 는 HTML5 최종 표준안을 발표
